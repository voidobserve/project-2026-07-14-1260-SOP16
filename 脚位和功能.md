flowchart TD
    subgraph 系统启动与初始化
        Start([系统上电/复位]) --> Disable_WDT[关闭看门狗]
        Disable_WDT --> Sys_Init[系统时钟与基础外设初始化 system_init]
        Sys_Init --> Disable_Debug[关闭HCK/HDA调试引脚映射]
        Disable_Debug --> UART0_Init[串口通信初始化 uart0_config]
        UART0_Init --> ADC_Init[ADC引脚与模块初始化 adc_pin_config / adc_config]
        ADC_Init --> Timers_Init[定时器初始化 tmr0 / tmr1 / timer2 / timer3]
        Timers_Init --> PWM_Init[PWM输出初始化 pwm_init]
        PWM_Init --> Fan_Init[风扇控制初始化 fan_ctl_config]
        Fan_Init --> GPIO_Init[状态与控制引脚初始化]
        GPIO_Init --> Var_Init[占空比限幅与模式变量初始化]
        Var_Init --> Power_On[开机缓启动处理 power_on_handle]
    end

    subgraph 主循环控制
        Power_On --> Main_Loop
        Main_Loop{主循环 while 1} --> Temp_Scan[温度扫描与过热检测 temperature_scan]
        Temp_Scan --> Fan_Scan[风扇状态检测与异常限幅 fan_scan]
        Fan_Scan --> Set_Duty[综合计算并设定目标脉宽 set_duty]
        Set_Duty --> Adj_Ch0[获取CH0平滑调节占空比 adjust_pwm_channel_0_duty]
        Adj_Ch0 --> Adj_Ch1[获取CH1平滑调节占空比 adjust_pwm_channel_1_duty]
        Adj_Ch1 --> Main_Loop
    end

    subgraph 串口通信处理
        UART0_IRQ[UART0接收中断] --> RX_Buffer[接收数据写入缓冲区 uart0_rx_buffer]

    end

    subgraph 定时器与中断调度
        TMR0_IRQ[Timer0中断] --> Task_Sched_0[基础时间片调度]

        TMR2_IRQ[Timer2中断] --> PWM_Ramp[PWM渐变与呼吸效果控制]
        TMR3_IRQ[Timer3中断] --> Engine_Sample[引擎信号采样触发]
        ADC_IRQ[ADC转换完成中断] --> ADC_Update[更新ADC采样值 adc_val]
    end

    subgraph 采样与控制闭环
        ADC_Update --> Read_Temp[读取热敏电阻电压 adc_val_from_temp]
        ADC_Update --> Read_Fan[读取风扇反馈电压 adc_val_from_fan]
        ADC_Update --> Read_Engine[读取引擎信号电压 adc_val_from_engine]
        ADC_Update --> Read_Knob[读取旋钮电压 adc_val_from_knob]
        
        Read_Temp --> Temp_Overheat[过热累计与状态判定]
        Read_Fan --> Fan_Err_Check[风扇堵转/异常判定]
        Read_Engine --> Engine_Filter[引擎信号滤波与过驱动判定]
